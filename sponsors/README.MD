Add .jpg or .png of match day sponsors which can be added to OBS on gameday.

For the automated weekend-sponsor graphic (shown at the end of each over, after the over
summary/partnership/AI commentary sequence): name the image file by an ID, e.g. `3.png`,
then set that same ID as "Weekend sponsor image ID" in the control panel, along with a
"Weekend sponsor name". Leaving the name blank in the control panel turns the feature off.
