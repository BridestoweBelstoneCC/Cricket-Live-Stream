Everything your stream needs

All the graphics you see on broadcast cricket, driven automatically by your scorer's laptop.

Live scorebar

Always-on bottom bar with score, overs, both batters, bowler figures, run rate, and a ball-by-ball ticker — updates on every delivery.

Player cards

Photo and season batting average slide in when a new batter arrives, pulled live from PlayCricket for both your side and the opposition.

Instant replays

Wickets, boundaries, and milestones replay automatically via the OBS replay buffer, with a full-screen transition.

AI commentary

Optional Sky Sports-style analysis at the end of every over, generated live by Claude AI from the actual match situation.

Result posts

Generate a polished Instagram result graphic for any match — home or away — with top batter, bowler, and match-day sponsors.

Phone control panel

Responsive control panel runs on any device over Wi-Fi. Manage the stream from your phone on the boundary.

Ball-by-ball database

Every delivery is logged to a local SQLite file as you stream — your own season-long dataset, exportable to CSV.

Full innings scorecard

Broadcast-style card at the innings break with all batters, dismissals spelled out in full, and bowling figures.

DLS par & weather

Live Duckworth-Lewis par score when rain is forecast. Drinks-break weather widget from Open-Meteo.

How it works

Three pieces of free software, already working together in minutes.

1

Scorer uses NV Play as normal

The scoring software writes a file on every ball. Nothing changes for the scorer.

2

The server reads it and processes the data

A small Python server running on the streaming laptop reads the scorer's file every few seconds, parses the ball data, and serves it to the overlay.

3

OBS mixes the overlay with your camera

The overlay is a browser source inside OBS. It polls the server and renders all the graphics — OBS sends the combined picture to YouTube.

4

Manage everything from the control panel

Open http://localhost:5000/control in any browser — on the streaming laptop or your phone over Wi-Fi — to set opposition details, toggle graphics, and run replays.

What you need

All software is free. You probably already have most of this.

Hardware

  • Windows or macOS laptop
  • Camera (USB webcam or HDMI)
  • Internet connection

Software (free)

  • OBS Studio
  • Python 3
  • NV Play / PCS Pro

Optional

  • PlayCricket API key (fixture detection)
  • Anthropic API key (AI features)
  • FFmpeg (highlights reel)
